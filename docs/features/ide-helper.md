# IDE helper integration

Reads property and method types from `_ide_helper_models.php` files generated by [Laravel IDE Helper](https://github.com/barryvdh/laravel-ide-helper), making them available to PHPStan for model attribute and accessor resolution.

## Configuration

| Option           | Default | Description                                     |
|------------------|---------|-------------------------------------------------|
| `typeIdeHelper`  | `true`  | Enable IDE helper integration                   |
| `ideHelperPath`  | `''`    | Custom path to IDE helper file (auto-detected if empty) |

## How it works

The extension parses `@property`, `@property-read`, and `@method` PHPDoc tags from the IDE helper file and exposes them to PHPStan:

```php
// _ide_helper_models.php
/**
 * @property string $id
 * @property Carbon|null $sent_at
 * @property-read string $display_name
 * @method static Builder|Post query()
 */
class Post extends \Eloquent {}
```

Properties marked `@property-read` are read-only — PHPStan will report errors if you try to assign to them.

The extension only provides properties and methods that aren't already declared on the real model class, avoiding conflicts with existing PHPDoc or actual class declarations.

## Path resolution

By default, the extension looks for `_ide_helper_models.php` in the project root. Set `ideHelperPath` to override:

```neon
parameters:
    filamentPhpstan:
        ideHelperPath: 'storage/_ide_helper_models.php'
```
